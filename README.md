# arduino-safe

A virtual safe using the Arduino Uno.

---

## Required Arduino Components
 - 1x Arduino Uno
 - 1x Breadboard
 - 4x Buttons
 - 4x LEDs
 - Resistors
 - Wires

In order to compile sketches and upload code for the Ardunio, the Ardunio IDE is needed. It can be downloaded [here](https://www.arduino.cc/en/Main/Software).

The Arduino Documentation can be found [here](https://www.arduino.cc/en/Reference/HomePage).
